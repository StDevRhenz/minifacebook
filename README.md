# minifacebook
This project is ideal for those learning CRUD operations, authentication, and client-server architecture.

# About Mini Facebook

**Users can:**

- **Sign up / log in**
- **Create a post (with image & caption)**
- **View others' posts**
- **Delete their own post**

## Technologies Used

- Next.js
- TypeScript
- Prisma (with Supabase)
- Authentication
- File Upload handling
